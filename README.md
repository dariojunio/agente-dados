# Data Agent

A conversational data analysis agent built with Streamlit and the OpenAI API. Upload a CSV or Excel file, ask questions in natural language, and get instant analysis, live Python/pandas execution, and charts without writing code.

## Features

- Natural language chat with a data expert agent, responding in Brazilian Portuguese
- Exploratory data analysis: statistical summary, null values, column types, and previews
- Controlled pandas execution generated by the agent
- Chart generation: bar, line, histogram, pie, scatter, and box plots
- Quick suggestion buttons for common analysis prompts
- Supports CSV, XLSX, and XLS files

## Security Notice

- The agent can execute restricted Python/pandas snippets generated by the model.
- The execution path blocks imports, file/network access helpers, private attributes, and pandas read/write helpers, but it is still not a full sandbox.
- Avoid uploading highly sensitive data such as medical records, secrets, credentials, or personal identifiers.
- Data is stored only in memory during the Streamlit session and is not persisted by this app.
- Store API keys in `.streamlit/secrets.toml` or environment variables, never in source code.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/dariojunio/data-analysis-agent.git
cd data-analysis-agent
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Set up your OpenAI API key

Create `.streamlit/secrets.toml`:

```toml
OPENAI_API_KEY = "your-key-here"
# Optional. Defaults to gpt-5-mini.
OPENAI_MODEL = "gpt-5-mini"
```

Or set environment variables:

```bash
export OPENAI_API_KEY="your-key-here"
export OPENAI_MODEL="gpt-5-mini"
```

### 4. Run the app

```bash
streamlit run app.py
```

## Tech Stack

- [Streamlit](https://streamlit.io/) - web interface
- [OpenAI API](https://platform.openai.com/docs) - language model and function calling
- [Pandas](https://pandas.pydata.org/) - data manipulation
- [Matplotlib](https://matplotlib.org/) - chart rendering

## Deployment

Ready to deploy on [Streamlit Cloud](https://streamlit.io/cloud). Connect the repository and add `OPENAI_API_KEY` to the app secrets. Optionally add `OPENAI_MODEL` to override the default model.
